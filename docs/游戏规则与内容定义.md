# 游戏规则与内容定义

## 文档定位

- 本文是 Watcher 当前版本的玩家向规则、概念与内容权威定义文档。
- 本文只描述当前已经实装并对对局结果生效的规则，不讨论实现细节、客户端表现细节或历史方案。
- 架构、代码入口与数据流请阅读 `docs/arch/` 下的开发文档；如与本文冲突，以本文中的规则定义为准。

## 第一部分 规则与概念

### 1. 回合

- Watcher 按回合进行。任一时刻，只有当前行动玩家可以主动使用工具。
- 一个标准回合分为三个规则阶段：
  1. `回合开始`
     - 确定当前行动玩家。
     - 结算回合开始触发的技能、状态与相关效果。
  2. `行动阶段`
     - 玩家投掷本回合的骰子并获得对应工具。
     - 玩家在允许的条件下使用工具。
  3. `回合结束`
     - 结算回合结束阶段开始时触发的技能、状态与相关效果。
     - 若该阶段获得了可用工具，则玩家可以继续使用这些工具，或主动跳过该阶段。
     - 当该阶段工具全部使用完毕，或玩家选择跳过后，才真正切换到下一名玩家的回合。
- 在通常情况下，投骰包含两部分：
  - `移动骰`：获得【移动】工具，其步数等于投掷的点数。
  - `工具骰`：从基础六种工具中随机给出一种。
- 技能、地形与状态可以改变标准回合流程，例如：
  - 在回合开始额外授予工具；
  - 在回合结束阶段额外授予工具；
  - 修改本回合的移动方式；
  - 跳过工具骰/移动骰；
  - 直接跳过一个回合。

### 2. 工具与技能

- `工具` 是玩家在对局中主动使用的行动资源。
  - 工具需要玩家交互才能使用。
  - 工具既可以来自骰子，也可以来自角色技能、地形或其他特殊效果。
- `技能` 是角色自带的被动能力。
  - 技能不需要交互。
  - 技能的常见作用包括：授予工具、改写投骰结果、改变移动规则、施加或移除状态。
  - 技能有特定的触发时机。
区分：技能是被动，工具是主动；主动“技能”应该通过赋予工具实现。

### 3. 位移

- 本文中的“位移”指玩家在棋盘上的位置变化。
- 当前规则中，位移主要分为以下几类：

#### 3.1 平移

- `平移` 指角色沿某个方向逐格推进。
- 平移会逐格处理路径，因此会正常受到阻挡、经过触发、停留触发等规则影响。
- 大多数普通移动、击退、推动都属于平移。

#### 3.2 拉拽

- `拉拽` 指角色被沿某个方向强制拖动。
- 拉拽同样会经过路径上的格子，但其语义属于“被拖动”，而不是正常行走。
- 典型例子是钩锁把角色拉向某个位置，或把其他玩家朝自己拉回。

#### 3.3 飞跃

- `飞跃` 指角色直接越过中间路径，落到最终落点。
- 飞跃能无视大多数路径上的地形；只有最终落点会结算正常接触与停留规则。
- 典型例子是跳跃、火箭中心爆炸造成的击飞。

#### 3.4 传送

- `传送` 指角色直接出现在目标位置，不沿中间路径移动。
- 传送不产生普通经过路径，也不逐格触发途中效果。

#### 3.5 行动力

 - `行动力` 指一次位移的剩余距离，地形/状态等因素都可能会影响行动力。

#### 3.5 位移状态

- `主动位移` / `被动位移`：移动、飞跃等位移工具执行的是主动位移，由其他工具造成的推动（如篮球、火箭等）都视为被动位移
- `回合内` / `回合外`：区分位移发生的时间，和 `主动` / `被动` 无关

#### 3.6 撞击

- `撞击 X` 事件指角色以平移方式前进时，因被阻挡而无法继续前进，且仍有剩余 `行动力`；其中 `X` 等于发生阻挡时剩余的行动力。
- 撞击只由平移产生，不由拉拽、飞跃或传送产生。
- 投掷物击中实体时也会触发 `撞击` 事件。
- 一些地形会对 `撞击` 作出响应，例如拳击球、塔；投掷物和角色触发的撞击事件有不同的响应。

### 4. 玩家状态

- `玩家状态` 指会持续一段时间、并在后续阶段继续生效的效果。
- 状态可以来自技能、工具、投射物、地形或模式规则。
- 当前规则中，至少包括以下两种通用状态：

#### 4.1 眩晕

- `眩晕` 会使玩家的下一个回合被跳过。
- 被眩晕的玩家仍然会正常进入自己的下一个回合开始，再立即结束该回合；随后眩晕被清除。

#### 4.2 束缚

- `束缚` 会降低玩家随后获得的带有移动点数的工具效果。
- 当前束缚按层数生效：层数越高，削减的移动点数越多。
- 束缚会在其结算周期结束后被移除。

## 第二部分 现有内容

### 5. 基础六种工具

- 基础六种工具指 `工具骰` 当前可直接掷出的六种工具。
- 它们不包含“移动”“制动”“平衡”等特殊来源工具。

#### 5.1 【跳跃】

- 默认规则：沿所选方向飞跃 `2` 格。

#### 5.2 【钩锁】

- 默认规则：沿所选方向发射长度 `4` 的钩锁。
- 命中结果：
  - 命中墙体时，将自己拉到墙前一格；
  - 命中玩家时，将目标朝自己方向拉回。

#### 5.3 【篮球】

- 默认规则：
  - 射程为全场；
  - 默认可反弹 `1` 次；
  - 命中玩家时将其击退 `1` 格。

#### 5.4 【砌墙】

- 默认规则：在周围八格内选择一个空地块，生成一面耐久 `2` 的土墙。
- 砌墙只能放在合法空位上，不能直接覆盖玩家或其他占位物。

#### 5.5 【火箭】

- 默认规则：
  - 射程为全场；
  - 落点爆炸时，中心目标会被击飞 `3` 格（视为 `飞跃`）；
  - 爆炸周围的相邻玩家会被推动 `1` 格。

#### 5.6 【拳击】

- 默认规则：
  - 射程 `2` 格；
  - 命中玩家时，将其击退 `3` 格；
  - 命中墙体时，会把自己向反方向弹开 `3` 格。

### 6. 角色与技能

- 以下内容按当前角色注册表列出。

#### 6.1 罗素（Late）

- 技能：`制动替换`
- 规则定义：
  - 当罗素获得“移动”工具时，不再获得普通移动，改为获得相同点数的【制动】。
- 【制动】：朝一个方向平移1~x格，x为本工具的移动点数

#### 6.2 鹅哈哈（Ehh）

- 技能：`额外篮球`
- 规则定义：
  - 每个行动阶段开始时，额外获得一个【篮球】。

#### 6.3 领导（Leader）

- 技能：`部署钱包`
- 规则定义：
  - 每个回合结束阶段开始时，获得一个【放置钱包】工具。
- 【放置钱包】：在以自身为中心的 `5x5` 范围内选择一个合法地块放置钱包。该工具本身不再直接结束回合；若它是该阶段最后一个工具，则该阶段会在使用后自然结束。

#### 6.4 布拉泽（Blaze）

- 技能：`炸弹准备`
- 规则定义：
  - 回合开始时，获得【备弹】；
  - 若成功备弹，则本回合后续阶段直接结束；
  - 在下一个回合的行动阶段开始时，获得一个“投弹”工具。
- 补充说明：
  - 投弹会先选择目标格，再选择推动方向，将目标格上的玩家推开。

#### 6.5 芙兰迪（Volaty）

- 技能：`飞跃开局`
- 规则定义：
  - 回合开始时，获得【飞跃准备】
- 【飞跃准备】：本回合放弃投掷工具骰子，将投掷的移动骰改为等距离的飞跃

#### 6.6 常（Chain）

- 技能：`静止钩锁`
- 规则定义：
  - 如果常在回合外没有被移动过，则其下一个行动阶段开始时，获得一个长度 `3` 的【钩锁】。

#### 6.7 法真（Farther）

- 技能：`制衡储存`
- 规则定义：
  - 行动阶段开始时，获得一个【制衡】；
  - 若此前已经储存过移动点数，则同时取回对应点数的“移动”工具。
- 补充说明：
  - 制衡可以在“压缩本回合移动 1 点”和“把本回合移动全部转存到下回合”之间二选一。

#### 6.8 莫汀（Mountain）

- 技能：`回合末砌墙`
- 规则定义：
  - 每个回合结束阶段开始时，获得一个耐久 `2` 的【砌墙】。

#### 6.9 AWM（AWM）

- 技能：`AWM`
- 规则定义：
  - 每个行动阶段开始时，获得一次【狙击】。
- 补充说明：
  - 狙击会向所选方向发射子弹；
  - 命中的第一格玩家会获得等同于当前总移动点数的束缚。

## 备注

- 地形、召唤物、地图模式等内容如需单独成为玩家向权威文档，应在后续以同等口径补写，而不是把实现细节散落到架构文档中。
