当前原型已经覆盖这些主链路：

- 工具、角色、地形、召唤物的 shared 权威逻辑
- golden case CLI / Web 回放
- 大厅与房间流程
- 自由模式与竞速模式
- 主页资料卡、创建房间页和房间内联加入

## 2026-04-03 房间入口美术更新

- 主页改成居中大标题布局。
- 玩家资料卡现在直接维护：
  - 用户名
  - 棋子外观
- `加入房间` 改成主页内联展开输入框。
- `创建房间` 改成单独的地图选择页。
- 地图选择页通过左右切换和滑动切换地图，并直接展示 shared 地图缩略图、模式和简介。
